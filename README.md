# Simply WT challenge 

## Initialisation 
- if you're adding a clean DB file please add it in the backend folder and call it 'sws.db'

### Backend
    - Please start a redis server on port 6379
    - cd backend
    - Run 'npm install'
    - Run `npm start`

#### Testing 
    - npm test

### Frontend 
    - Start backend (it will load a static page otherwise)
    - cd frontend
    - Run 'yarn install'
    - Run 'yarn start'

#### Testing 
    - yarn test
